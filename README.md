\## 📈 Key Metrics



| Metric | Description |

|------|-------------|

| 💰 Revenue | Total sales generated |

| 📈 Profit | Net profit |

| 📊 Profit Margin | Profitability percentage |

| 📦 Orders | Total number of orders |

\## 🎨 Dashboard Sections



| Section | Description |

|--------|------------|

| 🏠 Overview | KPI metrics + category \& region summary |

| 📈 Revenue Intelligence | Sales trends over time |

| 🏆 Product Analytics | Top products by sales \& profit |

| 📊 Category Deep Dive | Category \& sub-category performance |

| 🌍 Regional Insights | Region-wise sales \& profit |

| 💡 Business Insights | Key findings \& recommendations |

\## 💡 Key Insights



\- 📈 Sales show consistent growth trends over time

\- 📉 High discounts negatively impact profitability  

\- 🌍 Certain regions outperform others significantly  

\- 📊 Technology category leads overall performance
